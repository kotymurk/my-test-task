# Тестовое задание: Виджет выбора тарифов

Адаптивный виджет с динамическими данными, реализованный в рамках тестового задания на позицию Frontend-разработчика.

## 🚀 Стек технологий
* **Framework:** Next.js (App Router)
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **Data Fetching:** Fetch API

## ✨ Основной функционал
1. **Интерактивный выбор**: Выделение активного тарифа и автоматический выбор "лучшего предложения" при загрузке.
2. **Маркетинговый таймер**: 
   * Обратный отсчет на 2 минуты.
   * Динамическое изменение стилей (мигание красным) при остатке менее 30 секунд.
   * **Логика истечения**: Автоматическое скрытие скидок и старых цен после обнуления таймера с плавными переходами.
3. **Валидация**: Проверка обязательного согласия с офертой перед покупкой с визуальной индикацией ошибки.
4. **Анимации**: Эффект мигания кнопки покупки и плавные CSS-переходы элементов интерфейса.

## 📱 Адаптив
Проект полностью адаптирован под мобильные устройства (от 320px до десктопа) согласно предоставленным макетам.
